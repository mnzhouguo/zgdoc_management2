# 2. github国内加速

  • 找到github相关域名对于IP,同时找到耗时最小的IP DNS查询open in new window

  • 将IP域名映射添加到本地hosts文件中 C:\Windows\System32\drivers\etc

对于要经常访问的网站，我们可以通过在hosts中配置域名和IP的映射关系，提高域名解析速度。由于有了映射关系，当我们输入域名计算机就能很快解析出IP，而不用请求网络上的DNS服务器
